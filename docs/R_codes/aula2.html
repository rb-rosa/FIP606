Carregar pacotes

O carregamento de pacotes pode ser feito pelo menu ou então com um comando no console. O fluxo básico da programação pode ser conferido neste link.

Abaixo demonstraremos como instalar e carregar um pacote. A instalação é feita através do menu: Tools > Install packages.

library(tidyverse)
library(gsheet)
